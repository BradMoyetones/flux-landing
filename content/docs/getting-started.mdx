Welcome to the **Flux** documentation. This is where you can find all the information you need to get started with Flux.

## Installation

```bash
npm install flux-ui
```

## Setup

```tsx showLineNumbers
import { FluxProvider } from 'flux-ui';

function App() {
  return (
    <FluxProvider>
      <YourApp />
    </FluxProvider>
  );
}
```
